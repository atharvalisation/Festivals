# 🎉 Festivals – Indian Festivals Android App

An Android application that helps users explore the rich cultural heritage of India through festivals celebrated across different states.

The application provides festival information categorized by states, a searchable festival directory, image gallery, and an interactive user experience designed for learning and cultural awareness.

---

## 📱 Features

### 🏛️ State-wise Festival Exploration
Browse festivals from various Indian states including:

- Maharashtra
- West Bengal
- Tamil Nadu
- Gujarat
- Punjab
- Goa
- Rajasthan
- Kerala

### 🔍 Festival Search
Quickly search and discover festivals.

### 📋 Festival Directory
View a complete list of festivals available in the application.

### 🖼️ Festival Gallery
Explore festival-related images and visual content.

### 🎞️ Auto-Scrolling Banner
Dynamic banner section showcasing featured festivals and cultural highlights.

### 📚 Educational Purpose
Learn about India's diverse traditions, celebrations, and cultural heritage.

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/atharvalisation/Festivals.git
```

### Open Project

1. Open Android Studio
2. Select **Open Existing Project**
3. Choose the cloned repository
4. Allow Gradle Sync to complete
5. Run the application on:
   - Android Emulator
   - Physical Android Device
